# Nassau Candy Distributor – Shipping Efficiency Dashboard

## 📌 Project Overview

This project analyzes nationwide shipping performance for Nassau Candy Distributor using data-driven route intelligence.

The system transforms raw order and shipment data into actionable operational insights, identifying efficient routes, bottlenecks, and performance gaps.

Built using:
- Python
- Pandas
- Streamlit
- Data Visualization

---

## 🎯 Objectives

- Measure shipping lead time across routes
- Identify top 10 and bottom 10 performing routes
- Detect geographic bottlenecks
- Compare ship mode efficiency
- Provide interactive filtering for stakeholders

---

## 📊 Key Performance Indicators (KPIs)

- Shipping Lead Time (Ship Date − Order Date)
- Average Lead Time per Route
- Route Volume
- Delay Frequency (% exceeding threshold)
- Route Efficiency Score (Normalized)

---

## 🗺 Dashboard Modules

### 1️⃣ Route Efficiency Overview
- Route performance leaderboard
- Lead time benchmarking

### 2️⃣ Geographic Bottleneck Analysis
- Regional performance comparison
- Congestion-prone states detection

### 3️⃣ Ship Mode Performance
- Standard vs Expedited shipping comparison
- Lead time distribution

### 4️⃣ Route Drill-Down
- State-level insights
- Order-level shipment timelines
